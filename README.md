# 8 Queens

Simple javascript doodle to find solutions to the
[Eight queens Puzzle](https://en.wikipedia.org/wiki/Eight_queens_puzzle) .

Requires Nodejs 18+.

## Usage

Run using the CLI:

```bash
./run
```

## Project structure

Linting and formatting via npm modules Prettier and Eslint

```bash
npm install
npm run lint
npm run format
```
